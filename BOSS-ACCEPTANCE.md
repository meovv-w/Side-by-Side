# 同路行老板验收清单

目标：证明方案中的 V1 用户端、商家端和运营端都有可完成的业务流程，不是静态页面或占位按钮。

## 测试准备

1. 在 `/home/lin/workspace/tongdao-v2/server` 运行 `npm run demo`。
2. 微信开发者工具导入 `/home/lin/workspace/tongdao-v2/miniprogram`。开发者工具会自动连接本机 `8790` API。
3. 商家后台打开 <http://127.0.0.1:8790/merchant/?preview=1>。
4. 运营后台打开 <http://127.0.0.1:8790/ops/?preview=1>。
5. 手机测试可扫描 `preview-qrcode.png`，但微信号必须是对应 AppID 的管理员、开发者或体验成员。

注意：扫码版使用手机本地演示数据；开发者工具版使用真实 API，能与两个后台共享状态。跨端完整联调优先在开发者工具中测试。

## 用户端完整验收

### 地图与安全

1. 点击未读进入消息 Tab。
2. 点击其他车队只看到队长，不应看到对方队员位置。
3. 点击个人司机进入主页，关注后出现在“我的关注”。
4. 点击路况选择“绕行导航”并转发车队群。
5. 点击安全 POI 导航或拨号。
6. 以队长身份确认补油/补电提醒。
7. 点“报路况”提交说明，在运营后台通过后重新加载地图应出现事件。
8. 按住 SOS 到 100%，确认后应生成高优先通知；无紧急联系人时必须明确提示未发送原因。

### 行程与聊天

1. 行程列表切换顺路率、时间、距离排序。
2. 队长依次标记出发、开始行进、完成。
3. 出发后成员申请退出，队长审批前该成员仍可聊天。
4. 队长移除成员后，该成员不能继续访问群聊。
5. 完成行程后成员仍可发言，也可主动退出永久群。
6. 消息页切换全部、车队、地点、私信；“忽略”会标记已读，“拉黑”后不能私信。
7. 地点话题可关注、发图；归档话题只能看历史。

### 拼团、券和订单

1. 拼团列表切换附近/热门排序，商品图片不能空白。
2. 详情显示阶梯价格、当前进度、头像、剩余时间和商家入口。
3. 分享到车队群、微信好友和朋友圈。
4. 重复购买同一团不会生成第二个有效订单。
5. 我的券包按可用/已用/过期分类，日期不能写死。
6. 可用券显示 `CP` 券码并可复制。
7. 商家核销券码后，用户券状态变为已使用，运营结算页出现独立券结算。

### 我的、增长和客服

1. 编辑头像、昵称、车型、车牌和签名，刷新后保留。
2. 修改发现权限、车队/营销消息、位置、哨兵和紧急联系人。
3. 认证流程选择行驶证、完成活体、提交后显示审核中。
4. 勋章墙区分已获得/待解锁，安全上报通过后解锁安全勋章。
5. 我的页显示同路值流水，用户主页显示轨迹。
6. 邀请页显示分享路径、二维码、记录、奖励进度和 7 天手机号兜底。
7. 下一趟保存草稿后可找 10km 内同路队伍，或发起正式组队。
8. 客服工单提交后在运营后台可见，运营回复后用户可读。

## 商家后台完整验收

1. 提交店铺资料和营业执照；已入驻商家修改资料会进入运营复核。
2. 发布商品，阶梯价输入 `1:48, 3:39.9, 6:34.9, 10:29.9, 20:28.8`。
3. 尝试让后档价格上涨，保存必须被拒绝。
4. 编辑库存并上下架，用户端只展示上架商品。
5. 订单筛选与详情完整可用。
6. 核销类型切换为“优惠券核销”，输入 `CP520001`。
7. 保存银行卡和微信分账接收方，页面只回显掩码卡号。
8. 启停拉新券、修改面额/库存并加入奖励合作商家池。
9. 救援服务类型、半径、电话和营业状态刷新后保留。
10. 考核、通知和联系运营均显示真实 API 数据。

## 运营后台完整验收

1. 搜索用户并查看等级、同路值和认证状态。
2. 打开待审核行驶证原图，通过或填写原因拒绝。
3. 打开营业执照，审核商家和资料变更，调整等级/佣金规则。
4. 干预拼团成功或失败；失败时相关订单进入退款逻辑。
5. 搜索订单并打开完整详情。
6. 审批退款，触发订单分账。
7. 对平台券核销记录填写财务付款流水号并完成独立结算。
8. 创建平台券，修改预算和单用户上限。
9. 查看邀请排行榜并发奖励；确认合作商家券同时发放。
10. 修改成长分值，归档/下架地点话题。
11. 审核安全上报，通过后地图显示且用户获得同路值。
12. 打开客服完整对话，回复并关闭工单。
13. 暂停/恢复救援服务商，检查操作审计记录。

## 必须真机确认

- 麦克风授权、录音发送和扬声器播放。
- 微信登录授权、手机号能力和邀请小程序码。
- 定位授权、地图缩放聚合、系统导航和拨号。
- 微信好友/朋友圈分享。

## 企业凭证到位后再验收

- 微信 JSAPI 真实支付、回调验签、退款、分账。
- 高德实时路线、天气、海拔、POI 和交通事件。
- 腾讯云短信、IM 跨设备实时消息和离线推送。
- OCR/活体、对象存储、正式 HTTPS 域名和生产数据库。

## 给 HR 的演示视频顺序

建议 7-10 分钟：地图图层和浮窗 -> 行程申请/队长审批 -> 群聊语音/地点聊天/私信规则 -> 拼团用券/订单 -> 商家订单和券核销 -> 运营退款/结算/资质/客服 -> 我的成长/邀请/安全。
