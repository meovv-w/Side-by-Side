「同道」自驾社交平台 V1 MVP 增强版开发方案（最终版）
一、版本概述
项目
内容
版本名称​
V1 MVP 增强版（最终版）
核心目标​
上线最小可用产品，验证“车主通过平台组队+拼团省钱+拉新裂变”的完整商业闭环
核心功能​
用户认证 + 行程管理 + 匹配系统 + 聊天系统 + 地图展示（含信息分层+跨车队社交+地点聊天室） + 拼团系统 + 支付分账 + 核销系统 + 商家后台 + 运营后台 + 用户成长体系 + 拉新裂变体系 + 券体系 + 下一趟行程 + 群聊永久保留
技术栈​
Uni-App / React Native + Node.js + MySQL + Redis + 高德地图SDK + 腾讯云IM + 微信支付
