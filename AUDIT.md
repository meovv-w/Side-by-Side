## 架构和安全结论

- 小程序离线演示与远端 API 共用同一业务表面；演示不是唯一实现。
- 服务端按认证、用户、行程、聊天、地图、交易、商家和运营分域。
- MySQL 是生产仓库，Redis 管理验证码/会话状态，内存仓库只用于测试和演示。
- WebSocket 提供本地实时推送，腾讯 IM 适配器提供生产跨设备能力。
- 生产启动会强校验凭证；支付回调验签、库存事务、银行卡加密、字段白名单、RBAC 和审计均已实现。
- 每分钟任务处理过期订单/拼团、自动退款、地点归档、交通话题、掉队和结算对账。
