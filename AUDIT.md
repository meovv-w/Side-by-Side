- v2 已采用统一 Node.js API、MySQL/Redis 和前后端契约，继续搬旧代码会破坏当前边界。

## 新架构

- 小程序：原生微信小程序，离线持久演示与远端 API 双适配。
- API：Fastify，按认证、用户、行程、聊天、地图、交易、商家、运营拆分。
- 数据：MySQL 8.4、Redis 7.4，内存仓库只用于测试和演示。
- 实时：WebSocket 本地推送和腾讯云 IM 生产适配器。
- 外部服务：微信支付/登录、高德、腾讯短信、身份认证、对象存储。
- 安全：生产配置强校验、JWT、权限边界、支付回调验签、银行卡 AES-GCM、库存事务、审计日志和公开字段白名单。
